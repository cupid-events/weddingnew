ייבוא אנשי קשר

שתפו אנשי קשר מהאייפון או מהאנדרואיד כקובץ, העלו אותו כאן — והטבלה תיבנה לבד.

📇
לחצו כאן לבחירת קובץ אנשי קשר
קובצי vCard (‏.vcf) — אפשר לבחור כמה קבצים יחד, או לגרור לכאן
איך מוציאים אנשי קשר מאייפון? 🍏
איש קשר בודד: פתחו את אפליקציית "אנשי קשר" → בחרו איש קשר → גללו למטה → "שתף איש קשר" → שמרו ל"קבצים" (או שלחו לעצמכם) → העלו את קובץ ה‑vcf כאן.

כל אנשי הקשר בבת אחת: היכנסו מהמחשב ל‑iCloud.com → אנשי קשר → בחרו הכול → גלגל השיניים → "Export vCard" → יתקבל קובץ ‎.vcf אחד עם כולם, והעלו אותו כאן.
איך מוציאים אנשי קשר מאנדרואיד? 🤖
איש קשר בודד: פתחו את אפליקציית "אנשי קשר" → בחרו איש קשר → לחצו על שלוש הנקודות (⋮) בפינה → "שיתוף" → בחרו לשמור לקבצים / Drive או לשלוח לעצמכם → העלו את קובץ ה‑vcf כאן.

כל אנשי הקשר בבת אחת: פתחו את אפליקציית "אנשי קשר" → "תיקון וניהול" (או הגדרות ⚙️) → "ייצוא לקובץ" (Export) → יישמר קובץ ‎.vcf אחד עם כל אנשי הקשר → העלו אותו כאן.

💡 במכשירי סמסונג: אנשי קשר → תפריט (☰) → ניהול אנשי קשר → ייבוא/ייצוא אנשי קשר → ייצוא.
